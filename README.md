## Description

Simple app allows users to check the personal news feed from the social network [vk.com](https://vk.com/).

Before using the service you have to login into your vk account.

## Dependencies

The application uses [Cocoapods](https://cocoapods.org) dependency manager to include follows libraries:
- [SwiftyVK](https://github.com/SwiftyVK/SwiftyVK) for user authorization

Make sure to install the dependencies and open Xcode workspace instead of the project file:

```
pod install
open VK\ Feed.xcworkspace
```
